# Bloomberg Product Suite

Bloomberg's Product Suite encompasses the complete portfolio of Bloomberg professional products including the Bloomberg Terminal, data products, analytics solutions, trading platforms, media, and technology infrastructure.

## APIs

- **Bloomberg Open API (BLPAPI)** - Core API for the Bloomberg data ecosystem
- **Bloomberg Data License** - Enterprise bulk data delivery
- **Bloomberg Anywhere** - Remote access service for Terminal capabilities
